# 🔐 Secure Password Manager (Tkinter GUI)

A simple desktop Password Manager built using Python and Tkinter.  
This application allows users to generate strong passwords, store credentials locally, and search saved website logins.

---

## 🚀 Features

- 🔑 Random secure password generator  
- 💾 Save website credentials to a local JSON file  
- 🔍 Search for saved passwords by website  
- 📋 Automatically copy generated password to clipboard  
- 🖥 Clean and minimal Tkinter GUI  

---

## 🛠 Tech Stack

- Python 3
- Tkinter (GUI framework)
- JSON (local data storage)
- pyperclip (clipboard handling)
- random (password generation)
